# ☕ Aroma Senja - Coffee & Eatery

**Aroma Senja** adalah proyek website statis sederhana yang dibangun untuk mempelajari dasar-dasar pengembangan web modern menggunakan HTML5, CSS3, dan JavaScript. Website ini merepresentasikan profil sebuah cafe roastery dengan fitur reservasi meja interaktif.

---

## 🚀 Fitur Utama

* **HTML5 Semantic:** Menggunakan struktur dokumen yang modern dan ramah SEO (`<header>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`).
* **Katalog Menu:** Daftar menu cafe yang disajikan menggunakan elemen tabel HTML.
* **Reservasi Interaktif:** Formulir reservasi dengan validasi input (Email, Number, Date).
* **Logika JavaScript:** * Penggunaan variabel (`let`, `const`) dan berbagai tipe data.
    * Implementasi *Arrow Function* untuk kalkulasi biaya.
    * Logika percabangan (*if-else*) untuk pengecekan kapasitas meja.
    * Output dinamis ke DOM (Document Object Model).
* **Desain Responsif:** Layout CSS yang rapi dan nyaman dipandang di berbagai ukuran layar.

---

## 🛠️ Teknologi yang Digunakan

* **HTML5:** Struktur konten.
* **CSS3:** Penataan gaya dan tata letak (Layouting).
* **JavaScript (ES6+):** Logika interaktif dan manipulasi data.
